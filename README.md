# Sistema de Busca Otimizado para Catálogo de Produtos - MegaStore

## 📖 Descrição
Este projeto implementa um sistema de busca e recomendação de produtos utilizando **Rust**, com foco em eficiência e escalabilidade.

## 🛠️ Tecnologias
- Rust
- HashMap (tabelas hash)
- Estruturas de grafos
- Cargo (gerenciador de pacotes)

## ▶️ Execução
```bash
cargo build
cargo run
